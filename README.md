# 📰 Fake News Detection System

A Machine Learning web application built using **Python**, **Scikit-Learn**, and **Hugging Face** to predict news articles as Fake or Real.

---

## 📌 Project Overview

This application predict fake news or Real news using Machine Learning.

---

## 🚀 Features

- Unnamed
- Title
- Text
- Labels (Fake / Real)
---

## 🛠️ Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- Hugging Face
- Joblib

---

## 🤖 Machine Learning Models

- Multinomial Naive Bayes ✅ (Final Model)

- Decision Tree (Compared)


The Multinomial Naive Bayes achieved the best performance and was selected for deployment.
